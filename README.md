## 📖 Overview

The Ice Cream Shop Chatbot is a lightweight web application that allows customers to interact with an AI assistant through a clean chat interface.

Customers can:
- 🍦 Ask about available ice cream flavors
- 💰 Check prices
- 📍 Get shop information
- ⏰ View opening hours
- 📞 Contact the shop
- ❓ Ask general questions

The frontend communicates with a Flask backend using a REST API.


## 💬 API Endpoint

### POST `/chat`

#### Request

```json
{
  "input": "What flavors do you have?"
}
```

#### Response

```json
{
  "response": "We have Chocolate, Vanilla, Strawberry, Mango and Butterscotch."
}
```
